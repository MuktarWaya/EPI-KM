# คู่มือการใช้งานสำหรับวิทยากร (Facilitator Guide)

## 1. ก่อนเริ่มการอบรม
1. เข้าใช้งานระบบสร้างรอบการสอนใหม่ (Create Session) กำหนดชื่อ รพ.สต. และวันที่อบรม
2. รับ QR Code เฉพาะ Session สำหรับฉายขึ้นจอภาพหรือแจกผู้ปกครอง
3. เปิดหน้า **Presenter View** สำหรับวิทยากรบนแท็บเล็ต/โน้ตบุ๊ก และเปิดหน้า **Audience Display** ฉายขึ้นโปรเจกเตอร์

## 2. ระหว่างการอบรม
1. **ช่วงลงทะเบียน & PreTest:** ให้ผู้ปกครองสแกน QR Code และทำแบบประเมิน KS Model Likert 8 ข้อ
2. **ช่วงการสอนสด (Live Teaching Mode):** ใช้ Presenter View เลื่อนสไลด์ (50 สไลด์) และดู Speaker Notes พร้อมกดเปิดกิจกรรมสดเพื่อแสดงผลโหวตบนจอหลัก
3. **ช่วงเกมเรียนรู้:** ให้ผู้ปกครองตอบเกมเขียว-เหลือง-แดง และ Myth/Fact ผ่านมือถือ

## 3. สิ้นสุดการอบรม
1. กดปุ่ม **ปิดรอบการสอน (Close Session)** เพื่อหยุดรับคำตอบ
2. Export ข้อมูลส่งมอบให้ รพ.สต.
