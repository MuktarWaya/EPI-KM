# คู่มือการส่งมอบข้อมูลให้ รพ.สต. (Handover Guide)

## 1. วัตถุประสงค์
เพื่อส่งมอบข้อมูลผู้เข้าอบรม ผลประเมินความกังวล (KS Model) รายการ Flag ที่ต้องติดตาม และความประสงค์นัดหมาย ให้แก่เจ้าหน้าที่ รพ.สต. และ อสม. นำไปปฏิบัติงานต่อในพื้นที่ได้อย่างไร้รอยต่อ

## 2. ไฟล์ที่จะได้รับจากการ Export
1. **Executive Handover Summary (PDF/Summary):** สรุปจำนวนผู้เข้าร่วม จำนวนเด็ก รายการ Flag ค้าง และนัดหมาย
2. **FollowUp List (CSV):** รายชื่อเด็ก เบอร์โทรศัพท์ หมู่บ้าน อสม. ผู้รับผิดชอบ และป้าย Flag เพื่อใช้ติดตามเยี่ยมบ้าน
3. **Multi-sheet Excel Workbook:** ข้อมูลการอบรมแบบละเอียดทุกตารางสำหรับจัดเก็บเป็นฐานข้อมูล รพ.สต.
